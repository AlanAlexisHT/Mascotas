Página de inicio

A través del formulario de envío pueden mandarse datos para generar una tarjeta con el nombre, el tipo de animal, una descripción breve y un enlace para una imagen.

Los datos se almacenan en un archivo tipo SQLite3.

Cada tarjeta cuenta con los botones para dar "Me gusta", "No me gusta", "Compartir", "Guardar" y "Eliminar" en formato de icono aunque solamente está implementada la funcionalidad de éste último.

Esta página está sujeta a cambios.
